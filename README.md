# Weather Application

A simple Weather Application built using JavaScript and REST API that displays real-time weather information for any city.

## Features
- Search weather by city name
- Display temperature, humidity, and weather condition
- Real-time data fetched from OpenWeather API
- Error handling for invalid city names
- Simple and responsive UI

## Tech Stack
- JavaScript (ES6)
- HTML5
- CSS3
- OpenWeather REST API

## Key Concepts Used
- REST API integration
- Fetch API with async/await
- DOM manipulation
- Error handling in JavaScript

## How to Run Locally
1. Clone the repository
2. Open `index.html` in any modern browser
3. Enter a city name and click **Get Weather**

## Purpose
This project was built to understand API integration, asynchronous JavaScript, and real-world data handling using REST APIs.
